# Интеграция улучшенного геокодера в проект MapsNews

## ✅ Что изменилось (v2.0)

### Старая архитектура (v1.0):
- ❌ Требовала поддержку JSON-справочника с ~250 улицами
- ❌ Не понимала падежи ("на улице Ленина" не находилось)
- ❌ Ломалась на неполноте списка
- ❌ Простой string find() без NLP

### Новая архитектура (v2.0):
- ✅ **Без базы данных улиц** — не нужно поддерживать список
- ✅ **NLP через Natasha** — понимает падежи и формулировки
- ✅ **Regex fallback** — резервный механизм для простых случаев
- ✅ **Проверка региона** — фильтрует результаты вне Архангельской области
- ✅ **Умное кэширование** — экономия API-запросов

## 🔧 Архитектура геокодера

```
Текст новости
    ↓
┌─────────────────────────┐
│  1. Natasha Extractor   │ ← NLP-извлечение адреса
└─────────────────────────┘
    ↓ (если не найдено)
┌─────────────────────────┐
│  2. Regex Fallback      │ ← Паттерны "ул. X, д. Y"
└─────────────────────────┘
    ↓
┌─────────────────────────┐
│  3. Yandex Геокодер     │ ← "адрес + Архангельская область"
└─────────────────────────┘
    ↓
┌─────────────────────────┐
│  4. Проверка региона    │ ← Фильтр: только Архангельск
└─────────────────────────┘
    ↓
┌─────────────────────────┐
│  5. Кэширование         │ ← geo_cache.json
└─────────────────────────┘
```

## 🚀 Как это работает

### 1. Natasha AddrExtractor

```python
# Было (v1.0):
if "улица ленина" in text.lower():
    return "улица ленина"

# Стало (v2.0):
addr_extractor(text)  # Понимает:
# - "на улице Ленина"
# - "улицы Ленина"
# - "ул. Ленина, д. 5"
```

### 2. Regex Fallback

Если Natasha не смогла извлечь адрес, используется regex:

```python
patterns = [
    r'(?:ул\.?\s+)([А-Я][а-я]+)(?:,?\s*д\.\s*(\d+))?',  # "ул. Ленина, д. 5"
    r'(?:пр\.?\s+)([А-Я][а-я]+)',                       # "пр. Троицкий"
]
```

### 3. Проверка региона

```python
# После получения координат проверяем компоненты адреса:
if component["kind"] == "province":
    if "архангельск" in component["name"].lower():
        return coords  # ✅ Правильный регион
```

## 📝 Использование

Геокодер автоматически используется в `main.py`:

```python
geocoder = SimpleGeocoder()
address, coords = geocoder.process_text(title, content)
```
