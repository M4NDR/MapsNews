# MapsNews

**MapsNews** — это современная фуллстек-платформа, которая автоматически собирает региональные новости, находит в них географические адреса, и отображает их на интерактивной карте. 

Проект реализован на связке **FastAPI (Python) + React (JS)** с фокусом на премиальный дизайн (Glassmorphism), умный и адаптивный мобильный интерфейс и надежную внутреннюю систему логирования.

## Ключевые возможности
- **Умный Парсинг**: Автоматическое скачивание и форматирование статей с новостных порталов. Продвинутая фильтрация убирает рекламные блоки и блок "Читайте также".
- **Интегрированный Геокодер**: Фоновый воркер с использованием NLP (Natasha), который извлекает названия улиц и домов из сырого текста, а затем получает для них точные координаты. 
- **Мобильная Версия (PWA-Ready)**: Отдельный отполированный мобильный интерфейс с удобной навигацией (Pills), компактными карточками без лишних картинок и скроллом.
- **Встроенная Админ-панель**: Полностью нативная и защищенная паролем панель прямо на главном экране сайта. Выводит полный журнал работы геокодера и парсера без анимаций, в строгом утилитарном стиле.
- **Адаптивная Сетка**: Идеальный CSS-Grid, который подстраивается под любой размер экрана от узких смартфонов до ультрашироких мониторов.

## Стек технологий
- **Backend:** Python, FastAPI, SQLite, BeautifulSoup4, Natasha, Uvicorn 
- **Frontend:** React, React Router, CSS Variables (Dark/Light mode)

## Запуск
Для стабильной работы рекомендуется запускать бэкенд и фронтенд серверы раздельно.

1. **Бэкенд:**
   ```bash
   cd backend
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   uvicorn main:app --reload
   ```

2. **Фронтенд:**
   ```bash
   cd frontend
   npm install
   npm start
   ```
