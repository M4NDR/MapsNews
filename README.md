# MapsNews

**MapsNews** — это современная фуллстек-платформа, которая автоматически собирает региональные новости, находит в них географические адреса, и отображает их на интерактивной карте. 

Проект реализован на связке **FastAPI (Python) + React (JS)** с фокусом на премиальный дизайн (Glassmorphism), умный и адаптивный мобильный интерфейс и надежную внутреннюю систему логирования.

## Ключевые возможности
- **Умный Парсинг**: Автоматическое скачивание и стемминг/обработка текстов с новостного портала. Исключение рекламных блоков. Улучшенная классификация по категориям на основе корней слов.
- **Интегрированный Геокодер**: Фоновый воркер с собственным кастомным анализатором регулярных выражений, который извлекает названия улиц, домов и популярных объектов (ТЦ, Парки, Стадионы). Включает систему защиты от обрывов SSL-соединений (ретраи).
- **Интерактивная Карта с Пагинацией**: Просмотр новостей прямо на карте с бесконечной подгрузкой старых событий по кнопке "Показать ещё" (Lazy-loading).
- **Мгновенный Гео-Фильтр**: Стильная плавающая кнопка для моментальной фильтрации любой ленты новостей, оставляющая только записи с точными координатами.
- **Мобильная Версия (PWA-Ready)**: Отдельный отполированный мобильный интерфейс с удобной навигацией, компактными карточками без лишних картинок и скроллом.
- **Встроенная Админ-панель**: Защищенная паролем панель прямо на главном экране сайта. Выводит полный журнал работы геокодера. Позволяет вручную сбросить координаты конкретной новости для повторного сканирования алгоритмом!
- **Адаптивная Сетка**: Идеальный CSS-Grid, который подстраивается под любой размер экрана от узких смартфонов до ультрашироких мониторов.

## Стек технологий
- **Backend:** Python, FastAPI, SQLite, BeautifulSoup4, Natasha, Uvicorn 
- **Frontend:** React, React Router, CSS Variables (Dark/Light mode)

## Запуск
Для стабильной работы рекомендуется запускать бэкенд и фронтенд серверы раздельно.

1. **Бэкенд:**
   ```bash
   cd backend
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   uvicorn main:app --reload
   ```

2. **Фронтенд:**
   ```bash
   cd frontend
   npm install
   npm start
   ```
